**PF Endless Runner Game (C++ / OpenGL)**
A 2D Endless Runner game built in C++ using OpenGL/GLUT as a Programming Fundamentals (PF) first semester project at FAST, NUCES. The game includes lane switching, obstacle avoidance, coin collection, score tracking, and gradually increasing difficulty.

**Features:**
Player movement with lane switching (left/right)
Random obstacle generation
Coin collection system
Lives system (game ends when lives reach zero)
Increasing speed and difficulty over time
High score saving using file handling
Pause and resume functionality
Restart option after game over
Simple 2D graphics using OpenGL
**Concepts Used:**
C++ fundamentals (loops, functions, arrays)
File handling using fstream
Random number generation
Collision detection
Game state management (menu, playing, pause, game over)
OpenGL/GLUT rendering
**How to Run:**
Install OpenGL/GLUT on your system

_Compile the project using:_
g++ Tetris.cpp -o game -lGL -lGLU -lglut

_Run the executable:_
./game
